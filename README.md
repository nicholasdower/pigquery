# <picture><img src="icon.svg" height="32px"></img></picture> PigQuery

## Intro

A Chrome Extension offering some BigQuery enhancements.

## Features

- Insert queries and query snippets. (Ctrl+⌘+I)
- Share links. (Ctrl+⌘+S)
- Open result cell data in external sites. (Alt+Click)
- Copy cell data to clipboard. (Alt+⌘+Click)
